Bento Miso Collaborative Workspace

Solving the problem of filling in many administrative gaps at a tiny non-profit with a physical space and membership to manage, this site allows Bento to provide a holistic user experience, both in the space and through the site, to its members without hiring more staff.

Role

Everything: Prototyping, design, content, data modeling, development and maintenance.

Tech

PHP framework Daimio on MySQL and MongoDB.

Stripe API, Mandrill API, Flickr API.

Features

  • Member management system, from application through approval, billing and churn management; member features such as resource bookings, event and ticketing management, skill matchmaker, profile and projects, calendar feeds and more.
  • Office manager features like check-ins, approval workflows, helpdesk integration and resource calendars.
  • Community features aggregating events, profiles and projects.

SummerWorks Adventure

A prototype commission from SummerWorks 2013 to “gamify” the festival planning experience. Rather than simply wedge game elements into a digital catalog, my collaborator and I built an actual game (and a story-driven game engine, Euphrates). The project was built in 2 days, including content production, music and illustrations.

Users play through an interactive narrative, trying to create a festival experience that balances social interaction and efficiency. Each choice impacts the player’s score, and their selections are summarized at the end of the virtual festival so they could be used as a real guide in advance of the festival.

Development included design of a tool the client used to write and edit storylines, set scoring rules and outcomes, test and debug specific games, and more.

Role

Game design, game engine development, HTML/CSS/JS implementation and testing.

Tech

JavaScript + jQuery, Node, MongoDB

Features

Stateful games, adaptive storylines based on player choices, scoring system, story editing tool.
